## Prerequisites

1. `sudo apt install docker-compose`

### Run the client
1. `yarn install`
2. `yarn start`

OR

- `docker-compose up`

### Run the server

1. `chmod +x run.bash`
2. `./run.bash`

OR

1. Use `./run.bash` for initializing the database
2. `docker-compose up`

## Stack
- React
- Flask
